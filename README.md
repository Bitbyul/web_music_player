# web_music_player

## 설명
> 웹 기반으로 동작하는 뮤직 플레이어입니다. 반응형 UI로 설계되어 PC, 모바일 환경에서 사용 가능하며 사용자 로그인, 플레이리스트, mp3 파일 추가, 싱크 가사보기 등의 기능을 지원합니다.
